# Дизайн-концепты Kurs

Папка содержит пять черновых визуальных направлений для нового UI после отката к MVP Stage 8.

Файлы:

1. `variant-1-tihiy-kabinet.png` - спокойный светлый рабочий стиль с зеленым акцентом.
2. `variant-2-zhurnal-pro.png` - более плотный табличный интерфейс для админки и журнала.
3. `variant-3-myagkaya-shkola.png` - мягкий, чуть более дружелюбный стиль для ученика и ежедневного обучения.
4. `variant-4-direktorskiy-kontrol.png` - строгий вариант с темной навигацией и светлой рабочей областью.
5. `variant-5-minimal-uchet.png` - минималистичный учетный интерфейс с малым количеством декоративных элементов.
6. `variant-6-chistiy-den.png` - компактный светлый интерфейс без пояснительных текстов.
7. `variant-7-temniy-kontur.png` - компактный вариант с темной боковой навигацией.
8. `variant-8-verhnyaya-navigatsiya.png` - вариант с верхней навигацией вместо постоянной боковой панели.
9. `variant-9-mobilniy-prioritet.png` - более крупный и мягкий вариант с акцентом на мобильный сценарий.
10. `variant-10-suhoy-uchet.png` - самый сухой компактный интерфейс для учета.

Исходный генератор макетов:

- `design-concepts.html`
- `design-concepts-compact.html`

Интерактивный концепт для проверки навигации и компонентов:

- `navigation-concept-v7-light.html`
- `navigation-concept-v7-light-preview.png`

Интерактивные варианты симметричной дизайн-системы для всех MVP-разделов:

- `symmetry-system-options.html`
- `symmetry-system-a-balance-preview.png`
- `symmetry-system-b-compact-preview.png`
- `symmetry-system-c-spacious-preview.png`

Эти файлы пока не являются реализацией интерфейса приложения. Они нужны, чтобы выбрать визуальное направление перед новой спецификацией и отдельной веткой дизайна.
